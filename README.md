# Skills

Helpful little things that improve my own workflows. Reusable agent skills for Cursor and similar AI tools.

## Install

Using the [skills CLI](https://skills.sh/docs):

```bash
npx skills add vnord/skills
```

## Agent guidance

To use the repo-tracked `AGENTS.md` as global guidance for agents and Claude Code:

```bash
./scripts/install-agents.sh
```
